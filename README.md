# node-red-axis-acs
Simplfies integration with Axis Camera Station (ACS)

## Features
- Server information
- List device inventory
- List recording
- Start/Stop Recording
- Add Bookmark
- Export recoring (MP4)
- Fire a trigger (configured in ACS)
- Add devices to server
